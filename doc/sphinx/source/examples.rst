

=============
Examples
=============

PyFMI comes with a range of different examples which can be a very useful source of information when implementing and solving your own problems. The examples are located in the examples folder of your PyFMI installation and can be imported and run with the following Python commands::

    from pyfmi.examples import *
    
    fmi_bouncing_ball.run_demo()
